About Me

A backend-focused developer with a strong interest in building reliable systems

I am a backend-focused developer improving myself especially in C#, ASP.NET Core and Microsoft SQL Server. I work on relational database design, T-SQL queries, CRUD operations, stored procedures, views, functions and triggers, while applying object-oriented programming principles to build clean and maintainable applications.

On the application side, I develop ASP.NET Core Web API projects with layered architecture, validation, authentication and authorization using JWT. I also use Entity Framework Core, Redis, Docker, Git and GitHub as part of modern backend workflows.

My goal is to create backend solutions that are not only functional but also understandable, secure and ready for real-world use.
